# Bildgalleriet

Ett responsivt bildgalleri byggt med HTML, CSS och JavaScript.

## Funktioner
- Kategorier och navigering
- Filtrering via taggar
- Sökfunktion
- Dark/Light theme
- JSON-baserad data
- Lazy loading och optimerade bilder
- Responsiv och tillgänglig layout

## Tekniker
- HTML (semantik)
- CSS (Grid, Flexbox)
- JavaScript (modulärt, DOM-hantering)

Projektet är en del av min personliga utvecklarportfolio.
